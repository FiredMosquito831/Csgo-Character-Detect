# Welcome to my YoloV8 repo

#### This repo is mostly about the 2 models i trained as a fun side project, one for csgo ai character detection and one for gta online thermal character detection

##### Both models work with realtime inference and local image analysis and were trained with YOLOv8 small and nano based on simple personal datasets
